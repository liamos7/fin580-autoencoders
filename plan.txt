"Autoencoder asset pricing models" by Shihao Gu, Bryan Kelly, Dacheng Xiu

Motivation:
Standard theory says returns should be explainable as a linear combination of factors with loadings.
Fama-French gives observable factors and fixed betas, IPCA's gives time-varying linear betas in firm characteristics.
Linearity is inaccurate --> non-linear model for betas in form of autoencoder!

Architecture: 
take in firm characteristics \vec{z}_{i,t-1} per stock as N x P matrix and give K loadings \vec{\beta}_{j,t-1} per stock, 
also take in N stock returns and give K factors \vec{f}_t from a single linear layer (explainability, portfolios)
Product these for vector of predicted returns
Minimize loss function: sum over each stock at each time of (r_{i,t} - \vec{\beta}_{i,t-1} \cdot \vec{f}_t)^2

Performance Improvements:
project the returns using the Z_{t-1} matrix to OLS beta - like return of a portfolio that longs that feature (managed portfolios)
l1 regularization
early stopping, batch normalization, ensemble averaging

Chen-Zimmerman (https://www.openassetpricing.com)
